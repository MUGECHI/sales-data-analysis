# Sales Data Analysis Project

## Overview
This project analyzes retail sales data to identify trends in revenue, customer behavior, and product performance.

## Dataset
- 1,000 transactions
- Time period: Jan 2023 â€“ Jan 2024
- Fields include: order_date, customer demographics, product category, quantity, and sales value

## Tools & Technologies
- MySQL (data cleaning, aggregation, views)
- Python (pandas, matplotlib)
- Metabase (interactive dashboards)
- Git & GitHub (version control)

## Key Analyses
- Total sales and transaction volume
- Monthly sales trends
- Sales by product category
- Sales by gender
- Top-performing customers

## Key Insights
- Electronics generated the highest total sales.
- Sales peaked in May and October.
- Female customers accounted for slightly higher total sales.
- Average order value was 456.
